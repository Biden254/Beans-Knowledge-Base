# Predicates Used in First-Order Logic Representation

We define the following predicates:

1. Infests(pest, bean)
   Meaning: pest infests bean.

2. Controls(pesticide, pest)
   Meaning: pesticide controls pest.

3. Safe(pesticide, bean)
   Meaning: pesticide is safe to use on bean.

4. NotSafe(pesticide, bean)
   Meaning: pesticide is unsafe for bean.

5. Effective(pesticide, bean)
   Meaning: pesticide effectively protects bean.

6. BroadSpectrum(pesticide)
   Meaning: pesticide controls more than one pest.

7. HardToManage(pest)
   Meaning: no pesticide is known to control this pest.
